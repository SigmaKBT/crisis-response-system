# 🚨 Rapid Crisis Response System

A real-time emergency management system for hospitality environments.

## 🔥 Features
- Real-time alerts (Socket.IO)
- Role-based login (Admin/Staff)
- Analytics dashboard (Chart.js)
- Sound alerts (Siren)
- Premium UI

## 🛠 Tech Stack
- Flask
- Socket.IO
- SQLite
- HTML/CSS/JS

## ▶️ Run Locally
pip install flask flask-socketio eventlet  
python database.py  
python app.py
